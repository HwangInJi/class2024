01. 1부터 100까지 출력하기(for문, while문, do while문)

1. for문은 for(초기값; 조건식; 증감값)순서로 진행된다.
2. while문은 초기값; 조건식; 실행문; 증감값 순서로 진행된다.
3. do while문은 초기값; do-실행문; 증감값; while-조건식; 순서로 진행된다. 

// 01. for문
{        
        for (let i = 1; i <= 100; i++) {     // 초기값; 조건식; 증감값
        console.log(i);                        // 실행문;
    }
}

// 02. while문
{
    let num = 1;                    // 초기값;

    while (num <= 100) {      // 조건식;
        console.log(num);      // 실행문;
        num++;                     // 증감값;
    }
}

// 03. do while문
{
    let num = 1;                   // 초기값;

    do {
        console.log(num);     // 실행문;
        num++;                    // 증감값;
    } while (num <= 100);   // 조건식;
}

02. 1부터 100까지 짝수만 출력하기(for문 + if문)

조건인 1~100까지인 수 중에서 짝수만 출력하고 싶을 경우, if조건문에 나눈 나머지를 의미는 %를 사용하여 조건식을 만들면 된다. 

{        
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {               
            console.log(i);
        }
    }
}

03. 1부터 100까지 짝수는 파란색, 홀수는 빨간색 출력하기

파란색, 빨간색이라는 색을 출력하는 조건을 추가할 경우 문자열인 span과 style 태그를 사용해 조건식을 if조건과 +로 결합해야 한다.
이 때, document.write()는 브라우저 화면에 출력된다는 것을 의미하며 화면 아래에 표시된 걸 확인할 수 있다. 

{        
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            document.write("" + i + ""); 
        } else {
            document.write("" + i + "");
        }
    }
}

04. 중첩 for문을 이용해서 구구단 출력하기

중첩 for문은 외부 for문과 내부 for문이 있다는 것을 인지해야한다.
따라서 외부 for문인 i는 2~9까지의 값이 각각 9번 반복되고, 내부 for문인 j는 1~9까지의 값이 순차적으로 반복된다. 

{        
    for (let i = 2; i <= 9; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + " * " + j + " = " + i * j);
        }
    }
}
결과 확인하기
1단~9단까지의 구구단 수식
2 * 1 = 2
...
9 * 9 = 81

05. 중첩 for문을 이용해서 8단 출력하기

8단만 구구단 수식으로 나오게끔 하기 위해선 외부 for문인 i는 시작과 끝을 8로 고정한 후, 내부 for문인 j만 1~9까지 1씩 증가되도록 설정하면 된다. 

{        
    for (let i = 8; i <= 8; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + " * " + j + " = " + i * j);
        }
    }
}
