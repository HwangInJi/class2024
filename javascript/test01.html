01. 1부터 100까지 출력하기(for문, while문, do while문)

설명란

// 01. for문
{        
        for (let i = 1; i <= 100; i++) {
        console.log(i);
    }
}

// 02. while문
{
    let num = 1;

    while (num <= 100) {
        console.log(num);
        num++;
    }
}

// 03. do while문
{
    let num = 1;

    do {
        console.log(num);
        num++;
    } while (num <= 100);
}
결과 확인하기
01. 1~100
02. 1~100
03. 1~100
