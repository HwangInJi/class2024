01. 변수 : 데이터 저장

let을 사용하여 x, y, z에 변수값을 입력합니다.
변수는 데이터를 저장하는 기능을 하며, 이 때 저장된 데이터는 재사용이 가능합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
