01. 변수 : 데이터 저장

let을 사용하여 x, y, z에 변수값을 입력합니다.
변수는 데이터를 저장하는 기능을 하며, 이 때 저장된 데이터는 재사용이 가능합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수값 x는 초기값인 100에서 300으로, y는 200에서 400으로, z는 "javascript"에서 "jquery"로 변경합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

+를 사용하여 변수값 x에는 300, y에는 400, z에는 "jquery"를 추가합니다.
이 때, 변수값은 변경과 추가가 가능합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";

    console.log("03_________");
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
400
600
javascriptjquery

04. 상수 : 데이터 저장 + 데이터 변경(X)

const를 사용하여 상수로 선언된 변수값은 변경하거나 추가할 수 없으므로 x, y, z값은 초기값 그대로 출력됩니다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    // x += 300;
    // y += 400;
    // z += "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200

05. 배열 : 데이터 저장(여러개) : 표현방법 1 : 배열 선언

배열 선언인 const arr = new Array();를 생성합니다.
인덱스 값인 []는 0부터 시작하며, 배열값 arr[0], arr[1], arr[2]를 각각 100, 200, "javascript"으로 저장하면 초기값 그대로 출력됩니다. 

{
    const arr = new Array();

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

06. 배열 : 데이터 저장(여러개) : 표현방법 2 : 배열 선언과 동시에 초기값 설정

배열 선언인 const arr = new Array();를 생성 후 Array값에 100, 200, "javascript"를 입력하면 초기값 그대로 출력됩니다. 

{
    const arr = new Array(100, 200, "javascript");

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기

200

08. 배열 : 데이터 저장(여러개) : 표현방법 4 : 리터럴 방식으로 배열 선언과 동시에 초기값 설정

가장 많이 사용되는 리터럴 방식의 배열 선언입니다.
상수로 선언된 배열값인 []안에 100, 200, "javascript"를 입력하면 나열된 순으로 인덱스가 자동 지정되며, 초기값 그대로 출력됩니다.

09. 배열 : 데이터 저장(여러개) : 표현방법 5 : push() 메서드를 이용한 배열 방법

push()를 이용해 안에 값을 넣으면 배열 초기값을 추가로 저장할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    arr.push(400);                  // push()를 이용해 배열 초기값 추가 저장

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
    console.log(arr[3]);
}
결과 확인하기
100
200
javascript
400

10. 배열 : 데이터 저장(여러개) : 표현방법 6 : 다차원 배열

아래는 배열 안에 배열이 있을 경우 값을 출력하는 예제입니다.
1. 각각의 []를 하나의 덩어리로 묶어 큰 덩어리인 []값 안에 두 개의 작은 덩어리인 []값이 있음을 먼저 생각해야 합니다.
2. 큰 덩어리 []값에서 배열 요소는 총 3개이며, 첫번째 배열 요소[0]는 100, 두 번째 배열 요소[1]는 [200, 300], 세 번째 배열 요소[2]는 ["javascript", "jquery"]가 됩니다.
2-1. 작은 덩어리인 두번째 배열 요소 [200, 300]에서 첫번째 배열 요소[0]는 200, 두번째 배열 요소[1]는 300이 됩니다.
2-2. 작은 덩어리인 세번째 배열 요소 ["javascript", "jquery"]에서 첫번째 배열 요소[0]는 "javascript", 두번째 배열 요소[1]는 "jquery"가 됩니다.
3. console.log()를 사용하여 배열값에 배열 요소를 순서대로 입력하면 결과에 해당되는 초기값이 출력됩니다. 

{
    const arr = [100, [200, 300], ["javascript", "jquery"]];

    console.log(arr[0]);            
    console.log(arr[1][0]);         
    console.log(arr[1][1]);         
    console.log(arr[2][0]);         
    console.log(arr[2][1]);        
}
결과 확인하기
100
200
300

11. 배열 : 데이터 저장(여러개) : 표현방법 7 : 배열구조분해할당

(예제 1) 배열 선언을 통해 arr값으로 100, 200, "javascript"를 지정합니다.
const []를 이용해 배열을 해체한 후 그 안에 a, b, c라는 각각의 변수를 만들어 arr값을 다시 할당해줍니다.

(예제 2) 배열 선언을 통해 arr값으로 100, 200, [300, 400]을 지정합니다.
const []를 이용해 배열을 해체한 후 그 안에 a, b, [x, y]라는 각각의 변수를 만들어 arr값을 다시 할당해줍니다. 

{
    (예제 1)
    const arr = [100, 200, "javascript"];
    const [a, b, c] = arr;

    console.log(a);                 
    console.log(b);                 
    console.log(c);                 
}

100
200

}

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼침연산자

배열 선언을 통해 각각의 arr1, arr2, arr3값을 지정합니다.
console.log()를 이용해 결과를 얻을 때 arr앞에 ...이 붙은 경우엔 결과값은 ,가 붙지 않은 상태로 도출됩니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];        // ...이 앞에 있으면 결과값에 ,가 안붙음

    console.log(arr1);                  
    console.log(...arr1);               
    console.log(...arr1, ...arr2);      
    console.log(...arr3);               
}
결과 확인하기
100, 200, javascript
100200javascript
100200javascript300400jquery
100200javascript500
