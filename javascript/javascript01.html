01. 변수 : 데이터 저장

let을 사용하여 x, y, z에 변수값을 입력합니다.
변수는 데이터를 저장하는 기능을 하며, 이 때 저장된 데이터는 재사용이 가능합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수값 x는 초기값인 100에서 300으로, y는 200에서 400으로, z는 "javascript"에서 "jquery"로 변경합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

+를 사용하여 변수값 x에는 300, y에는 400, z에는 "jquery"를 추가합니다.
이 때, 변수값은 변경과 추가가 가능합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";

    console.log("03_________");
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
400
600
javascriptjquery

10. 배열 : 데이터 저장(여러개) : 표현방법 6 : 다차원 배열

아래는 배열 안에 배열이 있을 경우 값을 출력하는 예제입니다.
1. 각각의 []를 하나의 덩어리로 묶어 큰 덩어리인 []값 안에 두 개의 작은 덩어리인 []값이 있음을 먼저 생각해야 합니다.
2. 큰 덩어리 []값에서 배열 요소는 총 3개이며, 첫번째 배열 요소[0]는 100, 두 번째 배열 요소[1]는 [200, 300], 세 번째 배열 요소[2]는 ["javascript", "jquery"]가 됩니다.
2-1. 작은 덩어리인 두번째 배열 요소 [200, 300]에서 첫번째 배열 요소[0]는 200, 두번째 배열 요소[1]는 300이 됩니다.
2-2. 작은 덩어리인 세번째 배열 요소 ["javascript", "jquery"]에서 첫번째 배열 요소[0]는 "javascript", 두번째 배열 요소[1]는 "jquery"가 됩니다.
3. console.log()를 사용하여 배열값에 배열 요소를 순서대로 입력하면 결과에 해당되는 초기값이 출력됩니다. 

{
    const arr = [100, [200, 300], ["javascript", "jquery"]];

    console.log(arr[0]);            
    console.log(arr[1][0]);         
    console.log(arr[1][1]);         
    console.log(arr[2][0]);         
    console.log(arr[2][1]);        
}

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼침연산자

배열 선언을 통해 각각의 arr1, arr2, arr3값을 지정합니다.
console.log()를 이용해 결과를 얻을 때 arr앞에 ...이 붙은 경우엔 결과값은 ,가 붙지 않은 상태로 도출됩니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];        // ...이 앞에 있으면 결과값에 ,가 안붙음

    console.log(arr1);                  
    console.log(...arr1);               
    console.log(...arr1, ...arr2);      
    console.log(...arr3);               
}
결과 확인하기
100, 200, javascript
100200javascript
100200javascript300400jquery

1. 각각의 {}는 하나의 덩어리로 묶어 큰 덩어리인 {}값 안에 한 개의 작은 덩어리인 {}값이 있음을 먼저 생각해야 합니다.
2. 큰 덩어리 {}값에서 객체 속성은 총 3개이며, 첫번째 객체 속성.a는 100, 두번째 객체 속성.b는 { x: 200, y: 300 }, 세번째 객체 속성.c는 "javascript"가 됩니다.
3. 작은 덩어리인 두번째 객체 속성 { x: 200, y: 300 }에서 첫번째 객체 속성.x는 200, 두번째 객체 속성 .y는 300이 됩니다.
4. console.log()를 사용하여 객체값에 객체 속성을 순서대로 입력하면 결과에 해당되는 초기값이 출력됩니다.

💡 {}안에 값을 입력할 땐 엔터를 친 후 각각의 객체 속성을 지정해야하며 속성마다 끝에 ,를 붙입니다. 

{
    const obj = {
        a: 100,
        b: { x: 200, y: 300 },
        c: "javascript"
    }

    console.log("객체 표현방법7")
    console.log(obj.a);         
    console.log(obj.b.x);      
    console.log(obj.b.y);     
    console.log(obj.c);               
}
결과 확인하기
100
200
300

20. 객체 : 데이터 저장(키와값) : 표현방법 8 : 객체 + 함수가 있는 방식

변수 : 데이터 저장 : 숫자, 문자, 함수(실행문), 객체, 배열, 특수값(undefined, null) : 자료형

함수가 함께 있는 경우, function(){}을 이용합니다.
함수의 {} 안에 console.log()를 입력하면 그 안에 있는 내용이 결과값으로 도출됩니다.

💡 함수가 포함된 방식에서 결과값을 도출할 땐 ()를 붙입니다. 

{
    const obj = {
        a: 100,
        b: [200, 300],
        c: "javascript",
        d: function () {
            console.log("javascript가 실행되었습니다.");
        },      // 함수
        e: function () {
            console.log(obj.c + "가 실행되었습니다.");      // c에 지정된 값인 javascript를 불러옴
        },
        f: function () {
            console.log(this.c + "가 실행되었습니다.");     // c에 지정된 값인 javascript를 불러옴
        }
    }

    console.log(obj.a);
    console.log(obj.b[0]);
    console.log(obj.b[1]);
    console.log(obj.c);
    obj.d();       
    obj.e();      
    obj.f();      
}

21. 객체 : 데이터 저장(키와값) : 표현방법 9 : 객체구조분해할당

객체 선언을 통해 obj값으로 100, 200, "javascript"를 지정합니다. const {}를 이용해 배열을 해체한 후 그 안에 a, b, c라는 각각의 변수를 만들어 obj값을 다시 할당해줍니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript",
    }

    const { a, b, c } = obj;

    console.log(a);
    console.log(b);
    console.log(c);     
}
결과 확인하기
100
200
javascript

22. 객체 : 데이터 저장(키와값) : 표현방법 10 : 객체펼침연산자

(내용 수정 예정) 

{
    const obj1 = { a: 100, b: 200, c: "javascript" }
    const obj2 = { ...obj1, d: "jquery" }
    const obj3 = { ...obj1, ...obj2 }

    console.log(obj1);      // {100 200 javascript}
    console.log(obj2);      // {100 200 javascript jquery}
    console.log(obj2.a);    // 100
    console.log(obj2.b);    // 200
    console.log(obj2.c);    // javascript
    console.log(obj2.d);    // jquery
    console.log(obj3);      // {100 200 javascript jquery}               
}
