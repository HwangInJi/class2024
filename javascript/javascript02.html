01. 변수 : 데이터 불러오기

let을 사용하여 x, y, z에 변수값을 입력합니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100
200
javascript

02. 상수 : 데이터 불러오기

const를 사용하여 x, y, z값을 입력한 후 상수화 합니다. 

{
    const x = 100, y = 200, z = "javascript";

}

200

04. 배열 : 데이터 불러오기 : length(개수)

length는 개수를 뜻하며, 결과값인 arr에 length를 입력하면 지정된 배열값의 개수를 알 수 있습니다.

05. 배열 : 데이터 불러오기 : for()문

for()문은 const arr = []로 지정된 배열 값들의 반복되는 결과를 한번에 불러오는 방법입니다.
1. for()문은 초기화를 해주어야하며, index의 약자인 i를 초기화해주기 위해 let의 값을 0으로 만들어줍니다.
2. 상수로 지정된 값을 9개이기 때문에 length를 사용해 해당 값의 개수보다 작은 개수만 출력하는 조건식을 만들어줍니다.
3. 이 때 i값은 [0]~[8]까지 1씩 값이 늘어나야하기 때문에 ++로 증가값을 만들어 줍니다.
3. 결과값을 실행할 땐 arr값에 인덱스값인 i를 입력합니다.

💡 for문 -> for(초기값; 조건식; 증감값) 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < arr.length; i++) {        
        console.log(arr[i])                       
    }
}

06. 배열 : 데이터 불러오기 : forEach()문

아래의 코드는 배열(arr)을 사용하여 메서드인 forEach()를 사용하는 예시입니다.
1. forEach 사용하여 배열인 arr의 각 요소에 대해 주어진 함수를 실행합니다. 여기서 함수는 각 요소(element)를 인자로 받아서 실행됩니다.
2. forEach는 인덱스(index)를 매개변수로 받는 함수를 제공할 수 있으며, 이 때 인덱스값이 순서대로 출력됩니다.
3. forEach는 배열 자체(array)를 매개변수로 받는 함수를 제공할 수 있으며, 이 때 결과값은 동일한 배열이 9번 반복하여 출력됩니다.

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];
    // 01
    arr.forEach(function (element) {               
        console.log(element);
    });

    // 02
    arr.forEach(function (element, index) {        
        console.log(index);
    });

    // 03
    arr.forEach(function (element, index, array) {   
        console.log(array);
    });
}

07. 배열 : 데이터 불러오기 : for of

for of에서 element는 반복되는 배열(arr)의 각 요소를 나타내는 변수입니다.
(let element of arr)이 부분에서 arr은 배열의 각 요소를 변수 element에 할당하는 부분으로,
반복문이 실행될 때마다 element 변수에는 arr 배열의 다음 요소가 순서대로 할당됩니다.

💡 for of는 배열의 길이나 인덱스를 직접 다룰 필요가 없어 간결하고 가독성이 좋은 코드입니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let element of arr) {
        console.log(element);
    }
}

08. 배열 : 데이터 불러오기 : for in

for in에서 element는 반복되는 배열(arr)의 각 요소를 나타내는 변수입니다.
for in은 객체의 열거 가능한 속성인 인덱스들을 반복할 경우 사용되기 때문에 console.log()를 통해 결과값을 도출할 경우,
현재 인덱스의 값이 element이므로 arr[element]로 입력해야 지정된 결과값이 나옵니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

          // 그냥 element를 쓰면 0 1 2 3 ... 으로 나오기때문에 결과값은 arr[element]로 입력해야 지정한 결과값으로 나온다.
    for (let element in arr) {
        console.log(arr[element]);
    }
}

09. 배열 : 데이터 불러오기 : map()

map()는 배열(arr)을 사용하는 메서드인 forEach()와 입력 방식이 동일합니다.
element를 인자값으로 받아 index, array를 매개변수로 받을 수 있으며, 주어진 함수에 따라 배열의 각 요소를 변환하고자 할 때 사용됩니다.
단, forEach와는 달리 배열형식으로 결과값이 도출됩니다. 

{
                    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];
        
                    //01
                    arr.map(function (element) {       
                        console.log(element);
                    });
        
                    //02
                    arr.map(function (element, index) {     
                        console.log(index);
                    });
        
                    //03
                    arr.map(function (element, index, array) {     
                        console.log(array);
                    });
}
