05. 배열 : 데이터 불러오기 : for()문

for()문은 const arr = []로 지정된 배열 값들의 반복되는 결과를 한번에 불러오는 방법입니다.
1. for()문은 초기화를 해주어야하며, index의 약자인 i를 초기화해주기 위해 let의 값을 0으로 만들어줍니다.
2. 상수로 지정된 값을 9개이기 때문에 length를 사용해 해당 값의 개수보다 작은 개수만 출력하는 조건식을 만들어줍니다.
3. 이 때 i값은 [0]~[8]까지 1씩 값이 늘어나야하기 때문에 ++로 증가값을 만들어 줍니다.
3. 결과값을 실행할 땐 arr값에 인덱스값인 i를 입력합니다.

💡 for문 -> for(초기값; 조건식; 증감값) 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < arr.length; i++) {        
        console.log(arr[i])                       
    }
}

06. 배열 : 데이터 불러오기 : forEach()문

아래의 코드는 배열(arr)을 사용하여 메서드인 forEach()를 사용하는 예시입니다.
1. forEach 사용하여 배열인 arr의 각 요소에 대해 주어진 함수를 실행합니다. 여기서 함수는 각 요소(element)를 인자로 받아서 실행됩니다.
2. forEach는 인덱스(index)를 매개변수로 받는 함수를 제공할 수 있으며, 이 때 인덱스값이 순서대로 출력되며 인덱스는 0부터 시작합니다.

        console.log(index);

    // 03

        console.log(array);
    });

07. 배열 : 데이터 불러오기 : for of

for of에서 element는 반복되는 배열(arr)의 각 요소를 나타내는 변수입니다.
(let element of arr)이 부분에서 arr은 배열의 각 요소를 변수 element에 할당하는 부분으로,

{

}

10. 배열 : 데이터 불러오기 : filter()

아래의 코드는 배열(arr)을 사용하여 메서드인 filter()를 사용하는 예시입니다.
1. filter를 사용하여 배열인 num값의 각 요소에 대해 주어진 함수를 실행합니다. 여기서 함수는 각 요소(element)를 인자로 받아서 실행됩니다.
2. filter는 인덱스(index)를 매개변수로 받는 함수를 제공할 수 있으며, 이 때 인덱스값이 순서대로 출력되며 인덱스는 0부터 시작합니다.

12. 배열 : 데이터 불러오기 : 배열 펼침 연산자(Spread Operator)

아래는 다양한 배열펼침연산자 방법들의 예시입니다.
단, 04 예시처럼 ...이 붙은 문자열을 배열로 변활할 경우 단어가 하나씩 쪼개져서 결과값이 출력됩니다. 

{
    // 01 배열 복사
    const arr = [100, 200, "javascript"];
    const copy = [...arr];

    console.log(arr);      
    console.log(...arr);   
    console.log(copy);     
}

{
    // 02 배열 합치기
    const arr1 = [100, 200];
    const arr2 = [300, 400];
    const merged1 = [...arr1, ...arr2];
    const merged2 = [...arr1, "javascript", ...arr2, "jquery"];

    console.log(arr1.concat(arr2));    
    console.log(merged1);              
    console.log(merged2);              
}

{
    // 03 함수 인자로 사용하기
    function numbers(a, b, c) {
        return a + b + c;
    }

    const nums = [100, 200, 300];
    console.log(numbers(...nums));      
}

{
    // 04 문자열 배열로 변환하기
    const text1 = "javascript";
    const text2 = [...text1];

    console.log(text1);    
    console.log(text2);    
}

19. 객체 : 데이터 불러오기 : for in

for()문은 const obj = {}로 지정된 객체 값들의 반복되는 결과를 불러오는 방법입니다.
만약, a, b, c와 같은 문자와 같이 결합해서 불러오고 싶을 땐 +를 사용해줍니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    //01
    for (let element in obj) {
        console.log(obj[element]);
    }

    //02
    for (let element in obj) {
        console.log(element + ":" + obj[element]);     
    }
}
결과 확인하기
01.
100
200
javascript

02.
a: 100
b: 200
c: javascript

20. 객체 : 데이터 불러오기 : 객체구조분해할당

아래는 다양하게 객체 구조를 분해하고 할당하는 방법들의 예시입니다. 

{
    // 01 기본 분해 할당
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    };

    const { a, b, c } = obj;

    console.log(a);     
    console.log(b);     
    console.log(c);     
}

{
    // 02 기본값 설정
    const obj = {
        a: 100,
        b: 200
    }
    const { a, b, c = "javascript" } = obj;

    console.log(a);    
    console.log(b);    
    console.log(c);    
}

{
    // 03 새로운 이름으로 변경
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const { a: x, b: y, c: z } = obj;

    console.log(x);    
    console.log(y);    
    console.log(z);    
}

{
    // 04 새로운 이름 변경 + 분해 할당
    const obj = {
        a: 100,
        b: { c: 200, d: "javascript" },
    }
    const { a, b: { c, d } } = obj;

    console.log(a);    
    console.log(c);    
    console.log(d);    
}

21. 객체 : 데이터 불러오기 : 객체펼침연산자

...이 붙은 obj도 객체 안에 결과값을 불러오고 싶을 경우
const에 obj가 아닌 merge1, merge2와 같은 새로운 객체명을 지정한 후 console.log(객체명);을 입력하면 됩니다. 

{
    const obj1 = { a: 100, b: 200 }
    const obj2 = { c: 300, d: 400 }
    const merge1 = { ...obj1, ...obj2 };
    const merge2 = { ...obj1, x: "javascript", ...obj2, y: "jquery" };

    console.log(obj1);       
    console.log(merge1);     
    console.log(merge2); 
}
결과 확인하기
{ a: 100, b: 200 }
{ a: 100, b: 200, c: 300, d: 400 }
{ a: 100, b: 200, x: "javascript", c: 300, d: 400, y: "jquery" }

22. 서버 데이터 불러오기 : XMLHttpRequest

XMLHttpRequest는 서버 데이터를 불러올 수 있는 메서드이지만 최근에는 잘 사용하지 않습니다. 

{
    var xhr = new XMLHttpRequest();
    xhr.open("GET", "https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json", true);

    xhr.onreadystatechange = function () {
        if (xhr.readyState === 4 && xhr.status === 200) {
            var responseData = xhr.responseText;
            var jsonData = JSON.parse(responseData);

            console.log(jsonData);
        } else if (xhr.readyState === 4) {
            console.error("데이터 불러오기 오류: " + xhr.status);
        }
    };

    xhr.send();
}
결과 확인하기
60개의 Array값

23. 서버 데이터 불러오기 : fetch API

fetch는 XMLHttpRequest와 동일한 기능을 하며, 서버 데이터를 불러올 때 가장 많이 사용하는 메서드입니다.
fetch는 then과 같은 문법을 이어서 사용할 수 있는 장점이 있으며, 만약 fetch를 활용한 메서드식에서 오류를 확인하고 싶을 경우에는 catch를 사용하면 됩니다. 

{
    fetch("https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json")
    .then(response => {
        return response.json();
    })
    .then(data => {
        console.log(data);
    })
    .catch(error => {                                       
        console.error("데이터 불러오기 오류:", error);
    })
}

60개의 Array값
