# vscode 단축키
설정 : Ctrl + ,   
사이드 바 : Ctrl + b   
줄 복사 : Art + Shift + 화살표  
다중 선택 : Art + Shift + 클릭   
다중 키워드 선택 : 키워드 드래그 + Ctrl + D
